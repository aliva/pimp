# PIMP

## install

add this line to your `.bashrc` file

```
source /path/to/pimp.sh
```

## commands

### init

```
pimp init
```

### activate

activate virtualenv from any path inside project

```
pimp activate
```

### run

run pythin script within virtualenv, you don't need call activate directly

```
pimp run <script.py>
```

### shell

run python shell

```
pimp shell
```
